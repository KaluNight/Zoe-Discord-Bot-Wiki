# Zoe Commands - List

### This list contains all commands you can use. Click on any link on the right to look at a more detailed site about a command.

## Important Commands

| Command | Explanation | Link |
| --- | --- | --- |
| `/setup` | With this command you will get information on **how to set up Zoe** on your Discord server . Useful if you need to re-setup Zoe. | [`**_setup_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/setup/) |
| `/language` | With this command you can **select the language Zoe speaks** on your Discord server. | [`**_language_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/language/) |
| `/config` | This command opens an interactive message in which you can configure Zoe navigating with reactions. | [`**_config_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/config/) |
| `/refresh` | By issueing the command the **infopanel** and each **leaderboard will be updated manually**. | [`**_refresh_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/refresh/) |
| `/help` | When issueing this command, Zoe will send you the help message in direct messages. | [`**_help_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/help/) |
| `/reset` | The command performs a reset for your server. It **resets the current configuration** of your server. Try using this command when something is not going right. | [`**_reset_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/reset/) |
| `/boost` | This command lets you boost the server you're using the command on to a premium server. | [`boost`](https://wiki.zoe-discord-bot.ch/en/commands/important/boost) |
| `/subscription` | This command shows you your current Zoe subscription status. | [`subscription`](https://wiki.zoe-discord-bot.ch/en/commands/important/subscription) |

## Channel Feature Commands

### infoChannel

| Command | Explanation | Link |
| --- | --- | --- |
| `/create infoChannel` | This command creates a new infoChannel in which Zoe can send information about players current games. | [`**_create infoChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/create/infoChannel/) |
| `/delete infoChannel` | Deletes the infoChannel of your server after a refresh. | [`**_delete infoChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/delete/infoChannel/) |
| `/define infoChannel` | Creates an infoChannel like the create command, except that the text channel already exists and is only defined. | [`**_define infoChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/define/infoChannel/) |
| `/undefine infoChannel` | Undefines the current infoChannel. This does not delete the text channel, Zoe will just not send anything in it anymore. | [`**_undefine infoChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/undefine/infoChannel/) |

### rankChannel

| Command | Explanation | Link |
| --- | --- | --- |
| `/create rankChannel` | This command creates a rank channel in which Zoe sends updates about ranks of registered players. | [`**_create rankChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/create/rankChannel/) |
| `/delete rankChannel` | Deletes the rankChannel of your server. | [`**_delete rankChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/delete/rankChannel/) |
| `/define rankChannel` | Creates an rankChannel like the create command, except that the text channel already exists and is only defined. | [`**_define rankChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/define/rankChannel/) |
| `/undefine rankChannel` | Undefines the current rankChannel. This does not delete the text channel, Zoe will just not send anything in it anymore. | [`**_undefine rankChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/undefine/rankChannel/) |

### clashChannel

| Command | Explanation | Link |
| --- | --- | --- |
| `/create clashChannel` | This command creates a new clash channel where Zoe can send information about the current and upcoming clash for a specific league account. | [`**_create clashChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/create/clashChannel/) |
| `/delete clashChannel` | Command to delete an existing clashChannel. | [`**_delete clashChannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/delete/clashChannel/) |
| `/clash refresh` | This command must be sent within a clashChannel in order for it to be refreshed. | [`**_clash refresh_**`](https://wiki.zoe-discord-bot.ch/en/commands/clash/refresh/) |
| `/clash analysis` | Sends an analysis of a team. If this command is sent within a clash channel, the region is not mandatory. | [`**_clash analysis_**`](https://wiki.zoe-discord-bot.ch/en/commands/clash/analysis/) |

### matchhistoryChannel

| Command | Explanation | Link |
| --- | --- | --- |
| `/create matchhistorychannel` | This command creates a matchhistory channel in which Zoe sends all played games of the selected players. | [`**_create matchhistorychannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/create/matchhistorychannel) |
| `/delete matchhistorychannel` | Deletes the matchhistory channel of your server. | [`**_delete matchhistorychannel_**`](https://wiki.zoe-discord-bot.ch/en/commands/delete/matchhistorychannel) |

## Player/Account/Team Commands

| Command | Explanation | Link |
| --- | --- | --- |
| `/register` | With this command, **players** on your server **can register themselves** for Zoe. | [`**_register_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/register/) |
| `/create player` | This command creates a new player with the given information. | [`**_create player_**`](https://wiki.zoe-discord-bot.ch/en/commands/create/player/) |
| `/delete player` | Deletes the mentioned player from the system in your server. | [`**_delete player_**`](https://wiki.zoe-discord-bot.ch/en/commands/delete/player/) |
| `/create team` | This command creates a new team with the given name. This allows grouping players in the info panel. | [`**_create team_**`](https://wiki.zoe-discord-bot.ch/en/commands/create/team/) |
| `/delete team` | Deletes the mentioned team from the system in your server. | [`**_delete team_**`](https://wiki.zoe-discord-bot.ch/en/commands/delete/team/) |
| `/add playerToTeam` | Adds the mentioned player to the named team. | [`**_add playerToTeam_**`](https://wiki.zoe-discord-bot.ch/en/commands/add/playerToTeam/) |
| `/remove playerFromTeam` | Removes the mentioned player from the named team. | [`**_remove playerFromTeam_**`](https://wiki.zoe-discord-bot.ch/en/commands/remove/playerToTeam/) |
| `/add account` | This command is used to connect another/secondary league of legends account with a player in discord. | [`**_add account_**`](https://wiki.zoe-discord-bot.ch/en/commands/add/account/) |
| `/remove account` | Removes the named account from the mentioned player. | [`**_remove account_**`](https://wiki.zoe-discord-bot.ch/en/commands/remove/account/) |
| `/banAccount` | Launches the Banned List Wizard. This Allows you (an lol account owner) to delete their account from other servers and add it to the banned list to make it impossible for others to add it. | [`**_banAccount_**`](https://wiki.zoe-discord-bot.ch/en/commands/other/banAccount/) |
| `/show players` | With this command you will get displayed **all players** on your server with **all their accounts**. | [`**_show players_**`](https://wiki.zoe-discord-bot.ch/en/commands/important/show-players/) |
