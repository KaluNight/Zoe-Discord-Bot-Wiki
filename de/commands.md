# Zoe Befehle - Liste

> Diese Liste enthält alle Befehle, die du verwenden kannst. Klicke auf einen Link auf der rechten Seite, um eine detailliertere Seite über einen Befehl zu sehen.

## Wichtige Befehle

| Befehl | Erklärung | Link |
| --- | --- | --- |
| `/setup` | Mit diesem Befehl erhältst du Informationen darüber, wie du Zoe auf deinem Discord-Server einrichten kannst. Nützlich, wenn du Zoe nach einem Reset neu einrichten musst. | [Hier klicken](/de/commands/important/setup/) |
| `/language` | Mit diesem Befehl kannst du die Sprache auswählen, die Zoe auf deinem Discord-Server spricht. | [Hier klicken](/de/commands/important/language/) |
| `/config` | Dieser Befehl öffnet eine interaktive Meldung, in der du Zoe für die Navigation mit Reaktionen konfigurieren kannst. | [Hier klicken](/de/commands/important/config/) |
| `/refresh` | Mit diesem Befehl werden die Infotafel und jede Rangliste manuell aktualisiert. | [Hier klicken](/de/commands/important/refresh/) |
| `/help` | Wenn du diesen Befehl ausführst, sendet Zoe eine Hilfemeldung mit hilfreichen Wiki-Seiten. | [Hier klicken](/de/commands/important/help/) |
| `/reset` | Der Befehl führt einen Reset für deinen Server durch. Er setzt die aktuelle Konfiguration deines Servers zurück. Versuche diesen Befehl zu verwenden, wenn etwas nicht richtig läuft. | [Hier klicken](/de/commands/important/reset/) |
| `/boost` | Mit diesem Befehl kannst du den von dir verwendeten Server auf einen Premium-Server boosten. | [Hier klicken](/de/commands/important/boost) |
| `/subscription` | Dieser Befehl zeigt dir den aktuellen Status deines Zoe-Abonnements an. | [Hier klicken](/de/commands/important/subscription) |

## Kanal-Feature Befehle

### Infokanal

| Befehl | Erklärung | Link |
| --- | --- | --- |
| `/create infochannel` | Dieser Befehl erstellt einen neuen Infokanal, in den Zoe Informationen über die aktuellen Spiele der registrierten Spieler senden kann. | [Hier klicken](/de/commands/create/infoChannel/) |
| `/delete infochannel` | Mit diesem Befehl wird der Infokanal deines Servers nach einer Aktualisierung gelöscht. | [Hier klicken](/de/commands/delete/infoChannel/) |
| `/define infochannel` | Mit diesem Befehl kannst du einen Infokanal wie mit dem Befehl /create erstellen, nur dass der Discord-Textkanal bereits existiert und nur definiert wird. | [Hier klicken](/de/commands/define/infoChannel/) |
| `/undefine infochannel` | Dieser Befehl hebt die Definition des aktuellen Infokanals auf. Der Textkanal wird dabei nicht gelöscht, Zoe sendet nur nichts mehr in diesem Kanal. | [Hier klicken](/de/commands/undefine/infoChannel/) |

### Rangkanal

| Befehl | Erklärung | Link |
| --- | --- | --- |
| `/create rankchannel` | Dieser Befehl erstellt einen Rangkanal, in dem Zoe Updates über die Ränge der registrierten Spieler sendet. | [Hier klicken](/de/commands/create/rankChannel/) |
| `/delete rankchannel` | Löscht den Rangkanaldeines Servers. | [Hier klicken](/de/commands/delete/rankChannel/) |
| `/define rankchannel` | Erzeugt einen Rangkanal wie der Befehl /create, nur dass der Textchannel bereits existiert und nur definiert wird. | [Hier klicken](/de/commands/define/rankChannel/) |
| `/undefine rankchannel` | Deaktiviert den aktuellen Rangkanal. Dadurch wird der Textkanal nicht gelöscht, Zoe sendet nur nichts mehr in diesem Kanal. | [Hier klicken](/de/commands/undefine/rankChannel/) |

### Clashkanal

| Befehl | Erklärung | Link |
| --- | --- | --- |
| `/create clashchannel` | Dieser Befehl erstellt einen neuen Clashkanal, in dem Zoe Informationen über den aktuellen und kommenden Clash für ein bestimmtes LoL-Konto senden kann. | [Hier klicken](/de/commands/create/clashChannel/) |
| `/delete clashchannel` | Verwende diesen Befehl, um einen bestehenden Clashkanal zu löschen. | [Hier klicken](/de/commands/delete/clashChannel/) |
| `/clash refresh` | Dieser Befehl muss innerhalb eines Clashkanals gesendet werden, damit dieser aktualisiert werden kann. | [Hier klicken](/de/commands/clash/refresh/) |
| `/clash analysis` | Dieser Befehl sendet eine Analyse eines Teams. Wenn dieser Befehl innerhalb eines Clashkanals gesendet wird, ist die Region nicht zwingend erforderlich. | [Hier klicken](/de/commands/clash/analysis/) |

## Spieler/Konten/Team Befehle

| Befehl | Erklärung | Link |
| --- | --- | --- |
| `/register` | Mit diesem Befehl können sich Spieler auf deinem Server für Zoe registrieren (muss zuerst aktiviert werden). | [Hier klicken](/de/commands/important/register/) |
| `/create player` | Dieser Befehl erstellt einen neuen Spieler mit den angegebenen Informationen. | [Hier klicken](/de/commands/create/player/) |
| `/delete player` | Mit diesem Befehl löschen Sie den genannten Spieler aus dem System auf deinem Server. | [Hier klicken](/de/commands/delete/player/) |
| `/create team` | Dieser Befehl erstellt ein neues Team mit dem angegebenen Namen. Dies ermöglicht die Gruppierung von Spielern in der Infotafel. | [Hier klicken](/de/commands/create/team/) |
| `/delete team` | Verwende diesen Befehl, um das genannte Team aus dem System deines Servers zu löschen. | [Hier klicken](/de/commands/delete/team/) |
| `/add playertoteam` | Dieser Befehl fügt den erwähnten Spieler zum genannten Team hinzu. | [Hier klicken](/de/commands/add/playerToTeam/) |
| `/remove playerfromteam` | Entfernt den genannten Spieler aus dem genannten Team. | [Hier klicken](/de/commands/remove/playerToTeam/) |
| `/add account` | Dieser Befehl wird verwendet, um einen anderen/sekundären League of Legends-Kontos mit einem Spieler in Discord zu verbinden. | [Hier klicken](/de/commands/add/account/) |
| `/remove account` | Entfernt das genannte Konto von dem genannten Spieler. | [Hier klicken](/de/commands/remove/account/) |
| `/banaccount` | Startet den Verbotsliste-Assistenten. Damit kannst du (ein LoL-Kontoinhaber) sein Konto von anderen Servern löschen und es zur Verbotsliste hinzufügen, um es für andere unmöglich zu machen, es hinzuzufügen. | [Hier klicken](/de/commands/other/banAccount/) |
| `/show players` | Mit diesem Befehl bekommst du alle Spieler auf deinem Server mit all ihren Accounts angezeigt. | [Hier klicken](/de/commands/important/show-players/) |
