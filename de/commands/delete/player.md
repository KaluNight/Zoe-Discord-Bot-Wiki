# /delete player

>### Information
>**Dieser Befehl löscht die erwähnten [Spieler](/de/terms/player) von Zoe auf deinem Server. Sie werden aus allen Zoe-Funktionen entfernt.**
>{.is-info}

>### Erforderliche Berechtigungen:
>Manage channels
>{.is-success}


>### Verwandte Befehle:
>-   [/create player](/de/commands/create/player/)

<br>
