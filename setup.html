Zoe Discord Bot Setup

Here you can find everything regarding Zoe's setup.

 

You want to use Zoe on your server? Here's how you can do it. There is also a tutorial on YouTube:

Click here!

Be aware that to set up and use Zoe you need the "manage channels" permission on your server (if you are not the administrator anyway).

 

Adding Zoe

You can add Zoe to your server by clicking on "Add the bot" on the homepage, on sites like top.gg or bots.ondiscord.xyz and here on the Wiki in the bottom left in the menu bar.

 

Authorizing Zoe

Now you have to log in to Discord if necessary, then select the server you want to add Zoe to and confirm Zoe's permissions. You may still have to solve a captcha and then you can start.

Setup

When Zoe is added for the first time, she sends a short message to the server's default channel.

Then the first setting follows. You should select the language that Zoe should speak on your server. (For more information see >language). You can navigate with the reactions and thrn confirm the language you want Zoe zo speak.

Directly after that, you have to select the standard League of Legends server region. Available are:

  • BR - Brazil
  • EUNE - Europe Nordic & East
  • EUW - Europe West
  • LAN - Latin America North
  • LAS - Latin America South
  • NA - North America
  • OCE - Oceania
  • RU - Russia
  • TR - Turkey
  • JP - Japan
  • KR - Republic of Korea

You can change this setting later in the cofiguration if you wish (see >config).

Now Zoe sends another message about what happens next. More on this in the next section.

 

Creating an infoPanel and adding players

 

Now we are ready to go. First of all, you create an infochannel or infopanel that is located in this channel. All the players you are about to create will be listed here with all their accounts.

To do this, use the command >create infoChannel nameOfTheChannel. For nameOfTheChannel, enter a name for the channel you want.

To create players you can use the command >create player @DiscordMentionPlayer (Region) (SummonerName) and to add your secondary account >add accountToPlayer (Region) (SummonerName). If the option is activated in the configuration, players can also add themselves with the command >register.

Aaaand you are good to go! For the other features you can click here: core features
