Zoe Commands - List

This list contains all commands you can use.

 

Important commands

 

>setup

With this command you will get information on how to set up Zoe on your Discord server . Useful if you need to re-setup Zoe.

⇒ Need more information about this command? Click here: >setup

 

>language

With this command you can select the language Zoe speaks on your Discord server.

⇒ Need more information about this command? Click here: >language

 

>config

This command opens an interactive message in which you can navigate with reactions. 

⇒ Need more information about this command? Click here: >config

 

>show players

With this command you will get displayed all players on your server with all their accounts.

⇒ Need more information about this command? Click here: >show players

 

>refresh

By issueing the command the infopanel and each leaderboard will be updated manually.

⇒ Need more information about this command? Click here: >refresh

 

>register

With this command, players on your server can register themselves for Zoe (this only works if the option is enabled in the configuration > see >config).

⇒ Need more information about this command? Click here: >register

 

>help

When issueing this command, Zoe will send you the help message in direct messages.

⇒ Need more information about this command? Click here: >help

 

>reset

The command performs a reset for your server. It resets the current configuration of your server. Try using this command when something is not going right.

⇒ Need more information about this command? Click here: >reset

 

Create commands

 

>create infoChannel nameOfTheNewChannel

This command creates a new InfoChannel where Zoe can send information about players.

⇒ Need more information about this command? Click here: >create infoChannel

 

>create player @DiscordMentionPlayer (Region) (SummonerName)

This command creates a new player with the given information.

⇒ Need more information about this command? Click here: >create player

 

>create team teamName

This command creates a new team with the given name. This allows grouping players in the info panel. 

⇒ Need more information about this command? Click here: >create team

 

>create rankChannel nameOfTheNewChannel

This command creates a rank channel where Zoe sends updates about ranks of registered players.

⇒ Need more information about this command? Click here: >create rankChannel

 

>create leaderboard

This command start the wizard for creating leaderboards. 

⇒ Need more information about this command? Click here: >create leaderboard

 

>create clashChannel nameOfTheNewChannel

This command creates a new clash channel where Zoe can send information about the current and upcoming clash for a specific league account.

⇒ Need more information about this command? Click here: >create clashChannel

 

Delete commands

 

>delete player @DiscordMention

Deletes the mentioned player from the system in your server.

⇒ Need more information about this command? Click here: >delete player

 

>delete infoChannel

Deletes the infoChannel of your server after a refresh.

⇒ Need more information about this command? Click here: >delete infoChannel

 

>delete team teamName

Deletes the mentioned team from the system in your server.

⇒ Need more information about this command? Click here: >delete team

 

>delete rankChannel

Deletes the rankChannel of your server.

⇒ Need more information about this command? Click here: >delete rankChannel

 

>delete leaderboard

Command to delete an existing leaderboard.

⇒ Need more information about this command? Click here: >delete leaderboard

 

>delete clashChannel

Command to delete an existing clashChannel.

⇒ Need more information about this command? Click here: >delete clashChannel

 

Add commands

 

>add playerToTeam @DiscordMention (teamName)

Adds the mentioned player to the named team.

⇒ Need more information about this command? Click here: >add playerToTeam

 

>add account @DiscordMention (Region) (SummonerName)

This command is used to connect another/secondary league of legends account with a player in discord. 

⇒ Need more information about this command? Click here: >add account

 

Remove commands

 

>remove playerFromTeam @DiscordMention

Removes the mentioned player from the named team.

⇒ Need more information about this command? Click here: >remove playerToTeam

 

>remove account @DiscordMention (Region) (SummonerName)

Removes the named account from the mentioned player.

⇒ Need more information about this command? Click here: >remove account

 

Stats commands

 

>stats profile

Zoe provides some information about the selected profile.

⇒ Need more information about this command? Click here: >stats profile

 

>stats predictRole

Zoe predicts the role of 5 given accounts when they play together.

⇒ Need more information about this command? Click here: >stats predictRole

 

>stats teamAnalysis

Zoe predicts the role of 5 given accounts when they play together with pick an ban recommendations.

⇒ Need more information about this command? Click here: >stats teamAnalysis

 

Define commands

 

>define infoChannel #channelMention

Creates an infoChannel like the create command, except that the text channel already exists and is only defined.

⇒ Need more information about this command? Click here: >define infoChannel

 

>define rankChannel #channelMention

Creates an rankChannel like the create command, except that the text channel already exists and is only defined.

⇒ Need more information about this command? Click here: >define rankChannel

 

Undefine commands

 

>undefine infoChannel

Undefines the current infoChannel. This does not delete the text channel, Zoe will just not send anything in it anymore.

⇒ Need more information about this command? Click here: >undefine infoChannel

 

>undefine rankChannel

Undefines the current rankChannel. This does not delete the text channel, Zoe will just not send anything in it anymore.

⇒ Need more information about this command? Click here: >undefine rankChannel

 

Clash commands

 

>clash refresh

This command must be sent within a clashChannel in order for it to be refreshed.

⇒ Need more information about this command? Click here: >clash refresh

 

>clash analysis (Region) (SummonerName)

Sends an analysis of the team. If this command is sent within a clash channel, the region is not mandatory.

⇒ Need more information about this command? Click here: >clash analysis

 

Other commands

 

>banAccount

Launches the Banned List Wizard. This Allows you (an lol account owner) to delete their account from other servers and add it to the banned list to make it impossible for others to add it.

⇒ Need more information about this command? Click here: >banAccount

 

>patchNotes

This command sends you the current PatchNotes which are in English.

⇒ Need more information about this command? Click here: >patchNotes
