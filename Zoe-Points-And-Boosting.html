FAQ - Frequently Asked Questions

What happens if you stop boosting a server?

The premium features of the server stop working and you get your Zoe Points back. You can use them again directly.

 

What happens if my server with premium features is not boosted anymore?

The premium features will stop working but your settings will not be deleted. So, if you don't change your configuration in the meantime, reboost your server will reactivate the premium features.

 

What happens if a server changes size and category while I boost it?

It depends on the number of points you have left, it can happen in two scenarios:

- Either you have enough Zoe Points left and the server will stay boosted.

- Or you don't have enough Zoe Points and you will be overloaded. In this case you will receive a message to warn you. Several options are available to you in this case. Unboost some servers, delete some League accounts or take a higher subscription.
