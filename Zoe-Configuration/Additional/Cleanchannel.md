# Clean Channel

>Create a channel in which all commands or all messages are deleted automatically.
>{.is-info}

> Click on the corresponding setting :one:in the configuration menu.
<img src="/config_additional_menu.png" width="60%" /> <br>
>The next thing you will see is the deactivation message with a description and reactions.
> If you want to disable the option react with :one:to delete all Zoe commands after a few seconds react with :two:or with :three: to delete all messages sent in the channel. To cancel the configuration process react with :x:.
<img src="/config_additional_1_cleanchannel_activation.png" width="60%" /> <br>
>Now you got to decide if you already have a command channel you want to define (react with :one:) or want to create a new channel (react with :two:).
<img src="/config_additional_1_cleanchannel_activation_2.png" width="60%" /> <br>
