Zoe Translation

Here you can learn about how Zoe is translated.

 

Zoe (and this Wiki in the future) is available in several languages. These are: English, French and German (complete). Aswell as Russian, Turkish, Italian, Portuguese and Spanish (partially).

All translations have been done by the community. If you notice a mistake, you can report it on the Discord server (in #language-feedback). You can find the link here.

If you would like to help finish a translation that is not yet complete or add a new language (that you can translate to) click on the link to the Discord server and message the team in #check-in-translator.
