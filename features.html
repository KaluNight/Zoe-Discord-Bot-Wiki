This list contains all features of Zoe.

 

InfoChannel with infoPanel

Zoe provides an information panel in a text channel with information about the players in a game, so you can immediately see who is available for a game.  

Game Card

Zoe provides information about current games such as champions played, SoloQ rank or winrate with a champion played. 

rankChannel

In the rankChannel it is possible to track the players' ranked progress.

leaderboards

In leaderboards (currently) up to 10 players can compare themselves in a selected category.

clashChannel

Zoe can perform an analysis of your own and opposing clash teams to help with pick and ban decisions.

stats Command

Zoe provides information about the selected profile, including the top 3 highest champions, a summary of all masteries, the last 3 games played, ranking statistics of the player and a chart with the 6 highest mastery champs.
