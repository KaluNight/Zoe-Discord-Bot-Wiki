Refresh mode

Information

The refresh mode is set automatically by Zoe, depending on how Zoe is loaded. When Zoe is in classic mode, the showed refresh rate is an estimation since she doesn't work technically with a refresh rate. Zoe refresh each servers as fast as she can.

 

Related commands/pages:

  • >refresh

 

Refresh With Game Detection 

When a registered user have Discord Presence enable, Zoe can see the start of a game and will trigger a refresh. This increase by a ton the reactivity of Zoe. This is why this is highly recommended to have Discord Presence enable if you want to have good result with Zoe.

Note : If you are sharing your screen with the built-in Stream feature of Discord, the game won't be detected. (Because it's show “Currently streaming League Of Legends” instead of “In game”).
