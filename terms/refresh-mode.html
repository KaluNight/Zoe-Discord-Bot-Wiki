Refresh mode

Information

The refresh mode or refresh rate is set automatically by Zoe, depending on how Zoe is loaded.  

 

Related commands/pages:

  • >refresh

 

Classic mode

When the load is low, Zoe remains in classic mode and updates everything by default rates. However, the refresh command always works. A refresh can be triggered by the detection of game with Discord Presence.

 

Smart mode

If many players are playing and the load is high, Zoe switches to smart mode and only updates the info panel (or sends the game cards) when it detects a game with a Discord presence (status) of a player. The refresh command still work in this mode.
