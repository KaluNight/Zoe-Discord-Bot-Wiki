# /add playerToTeam @DiscordUser teamName


>### Information: 
>**This command adds the mentioned player to the named team. For this, it must first be created and an infoPanel must exist on your server.**
>{.is-info}

>### Permissions needed: 
>Manage channels
>{.is-success}
