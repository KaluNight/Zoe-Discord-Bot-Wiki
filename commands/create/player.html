/create player @DiscordMention (Region) (Summonername)

 

Information

This command is used to create a player.  

To do this, you must enter the appropriate data IN THE BRACKETS (important) to link a League account. Insert the acronym for “Region” and the username of the account for “SummonerName”.  

Available Server Tags: BR, EUNE, EUW, JP, KR, LAN, LAS, NA, OCE, RU & TR

Permissions needed: 

none

 

Related commands:

  • /delete player
