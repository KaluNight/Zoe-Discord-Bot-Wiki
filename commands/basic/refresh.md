# /refresh
## Information
**When you execute the command your server is updated manually. More specifically, the cache is also updated (which normally happens every 24 hours).** Â <br>
**Refresh command refreshes:**
- [x] Infochannel
- [x] Rankchannel
- [x] Leaderboards
- [x] Matchhistorychannel
- [x] Rank roles
- [ ] Clashchannel (The [Clashchannel](/en/features/clashChannel) can be refreshed with the command [`/clash refresh`](/en/commands/clash/refresh/).)
<br>

### Permissions needed:
>**None** - every user on a server can use this command (as long as not further restricted by permission) {.is-success}
