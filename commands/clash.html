Clash commands

 

>clash refresh

This command must be sent within a clashChannel in order for it to be refreshed.

⇒ Need more information about this command? Click here: >clash refresh

 

>clash analysis (Region) (SummonerName)

Sends an analysis of the team. If this command is sent within a clash channel, the region is not mandatory.

⇒ Need more information about this command? Click here: >clash analysis
