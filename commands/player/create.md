# /create player @DiscordUser region gamename tag
## Information
**This command is used to create a player. Â To do this, you must enter the appropriate data for `@DiscordUser`, [`region`](/en/terms/region), [`gamename and tag (Riot ID)`](/en/terms/riotid) to link a League of Legends account to a member of your discord server. You can find all server region tags here: [Regions](/en/terms/region)**

**This command is meant to be for administrators that want to create the account for their players.**
<br>

### Permissions needed:
>**Manage channels** - only users with this permission ("moderators"/"administrators") can use this command as it affects Zoe for the server more deeply {.is-warning}
