# /undefine infochannel

>### Information
>**This command undefines the current [infochannel](/en/features/infoChannel). This does not delete the text channel, Zoe will just not send anything in it anymore.**
>{.is-info}

>### Permissions needed: 
>Manage channels 
>{.is-success}
