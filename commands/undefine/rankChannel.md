# /undefine rankchannel

>### Information
>**This command undefines the current [rankchannel](/en/features/rankChannel). This does not delete the text channel, Zoe will just not send anything in it anymore.**
>{.is-info}

>### Permissions needed: 
>Manage channels 
>{.is-success}
