Create commands:

 

>create infoChannel nameOfTheNewChannel

This command creates a new InfoChannel where Zoe can send information about players.

⇒ Need more information about this command? Click here: >create infoChannel

 

>create player @DiscordMentionPlayer (Region) (SummonerName)

This command creates a new player with the given information.

⇒ Need more information about this command? Click here: >create player

 

>create team (teamName)

This command creates a new team with the given name. This allows grouping players in the info panel. 

⇒ Need more information about this command? Click here: >create team

 

>create rankChannel nameOfTheNewChannel

This command creates a rank channel where Zoe sends updates about ranks of registered players.

⇒ Need more information about this command? Click here: >create rankChannel

 

>create leaderboard

This command start the wizard for creating leaderboards. 

⇒ Need more information about this command? Click here: >create leaderboard

 

>create clashChannel nameOfTheNewChannel

This command creates a new clash channel where Zoe can send information about the current and upcoming clash for a specific league account.

⇒ Need more information about this command? Click here: >create clashChannel
